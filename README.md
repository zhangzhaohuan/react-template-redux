## 首次运行
```
### 删除package-lock.json
### node sass 单独安装执行：
npm install node-sass@4.13.0 --registry=https://registry.npmmirror.com --sass_binary_site=https://npmmirror.com/mirrors/node-sass/ -D

### 安装依赖
npm i

### 启动项目
npm run start
```
